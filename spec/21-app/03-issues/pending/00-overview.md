# Pending Issues

_None at present._

When filing a new issue, create `{NN}-{kebab-name}.md` in this folder using the template in [`../00-overview.md`](../00-overview.md#file-template), then add a row to the **Pending Issues** table in that overview.

When the issue is resolved, **move** the file to `../solved/`, update its `**Status:**` line to `solved in v{X.Y.Z}`, set `**Resolved:** YYYY-MM-DD`, and re-link from `../00-overview.md` (move the row from Pending → Solved).
